===
Название: ncdu
Описание: Утилита для анализа использования дискового пространства в виде ncurses-интерфейса
Синтаксис: ncdu [опции] [каталог]
Пример: ncdu /
Примечание: Удобно искать папки/файлы, которые занимают много места

Название: htop
Описание: Интерактивный монитор процессов для Unix-систем
Синтаксис: htop [опции]
Пример: htop
Примечание: Улучшенная версия команды top с цветным интерфейсом и управлением с клавиатуры

Название: iostat
Описание: Отображает статистику ввода-вывода для устройств и разделов
Синтаксис: iostat [опции]
Пример: iostat -i
Примечание: В твоем примере isof -i похоже на опечатку, правильная команда для мониторинга ввода-вывода — iostat

Название: sudo !!
Описание: Выполняет последнюю команду с правами суперпользователя
Синтаксис: sudo !!
Пример: sudo !!
Примечание: Полезно, если забыл добавить sudo перед предыдущей командой

Название: tree
Описание: Отображает структуру каталогов в виде дерева
Синтаксис: tree [опции] [каталог]
Пример: tree -L 2 /home/user
Примечание: -L ограничивает глубину отображения

Название: wget
Описание: Утилита для скачивания файлов из интернета
Синтаксис: wget [опции] [URL]
Пример: wget -c http://example.com/file.iso
Примечание: Опция -c позволяет докачивать файл с места обрыва

Название: df
Описание: Показывает информацию об использовании дисков и файловых систем
Синтаксис: df [опции]
Пример: df -hT
Примечание: -h вывод в человеко-читаемом виде, -T показывает тип файловой системы

Название: at
Описание: Планировщик задач для выполнения команд в указанное время
Синтаксис: at [время]
Пример: echo "reboot" | at 02:00
Примечание: Для просмотра заданий используется команда atq, для удаления — atrm

Название: alias
Описание: Создает псевдонимы для команд
Синтаксис: alias имя='команда'
Пример: alias ll='ls -la'
Примечание: Упрощает использование длинных команд

Название: unalias
Описание: Удаляет псевдонимы команд
Синтаксис: unalias [опции] имя
Пример: unalias ll
Примечание: Для удаления всех псевдонимов — unalias -a

Название: cowsay
Описание: Выводит текст в виде «речевого облака» с ASCII-арт коровы
Синтаксис: cowsay [текст]
Пример: cowsay "Hello, world!"
Примечание: Можно менять персонажей (cowsay -f tux "Привет!")
